# Hashtag-Trend-Analytics-BigData
Project OverviewThis project focuses on analyzing hashtag trends and user engagement across social media platforms. It combines standard Python data processing with Big Data concepts like MapReduce and Apache Hive to demonstrate how large-scale datasets are handled and visualized.  ObjectivesTrend Analysis: Track hashtag popularity and evolution over time.  Engagement Metrics: Measure performance by calculating total engagement ($Likes + Retweets$).  Big Data Logic: Apply conceptual MapReduce phases (Map/Reduce) and HDFS distribution principles.  Interactive Visualization: Build a dashboard using Google Charts for real-time insights.  Technical StackLanguage: Python (Pandas, Seaborn, Matplotlib).  Big Data Tools: Apache Hive, HDFS.  Visualization: Google Charts Dashboard.  Core Components1. Python Data PipelineThe data is processed using a structured pipeline:Data Loading: Structured CSV datasets are loaded via pandas.  Cleaning: Conversion of date columns to datetime formats and sorting.  Metric Calculation: New columns are generated to track $Engagement = Likes + Retweets$.  2. MapReduce Implementation (Conceptual)The project simulates the MapReduce lifecycle manually in Python:Map Phase: Data is split into key-value pairs (e.g., #AI, 396).  Reduce Phase: Aggregates the mapped data to calculate total posts and engagement per hashtag.  3. Hive AnalyticsThe project includes a full Hive implementation for scalable querying:Database Setup: Creation of a hashtag_db and external tables in HDFS.  Data Analysis: SQL-like queries used to determine top-performing hashtags and daily trends.  Exporting: Inserting results into HDFS directories for further visualization.  Visual InsightsThe analysis produces several key visualizations:Line Plots: Showing hashtag trends over time.  Pie Charts: Representing the distribution of total posts by hashtag.  Boxplots: Highlighting the distribution of engagement.  Scatter Plots: Correlating the number of posts vs. the number of likes.  ConclusionThis project demonstrates the transition from local Python analysis to scalable Big Data environments. By integrating Hive and MapReduce concepts, it provides a blueprint for handling high-volume social media data efficiently.
